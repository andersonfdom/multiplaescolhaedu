Technology

We simply use Ajax technique instead of Web Sockets. Because it's pretty straightforward and works even on Internet Explorer. Also, you probably won’t need any server modifications for this plugin. It works with native PHP 5 server and WordPress well.

Why we don't use comets?

As you know comet is a web application model in which a long-held HTTP request allows a web server to push data to a browser, without the browser explicitly requesting it. This technique was also a good alternative, but we are more familiar with Ajax. And it works good enough for chat.

How to

How to create an operator?

It's brutally simple! Just like creating a normal user from administration panel.

  • Go to Users menu and click Add New link
  • Fill out  form
  • Find Role field and select Chat Operator in the list

Now this user CAN:

  • Chat with both operators and visitors
  • Read  and delete chat logs
  • Manage posts

How to go offline on chat?

Once you open chat console, you automatically connected to the chat. When you decide to go offline, just click Online button on the Chat Console page without logged out. It is good when you don't need to go offline for a long time.

How to customize skin?

If you only need to change colors of current skin, just open Options under the "Chat Console" menu and then you can change features from there.

If you are advanced user, you can create your own skin. Just skip next question:

How to create your own skin? 

This is for ADVANCED USERS.

  • Find basic folder inside "wp-contents/plugins/screets-chat/skins"
  • Clone this folder in the same place
  • Give a new name to this folder (such as my-skin)
  • Go to options page under Chat Console menu
  • Change "Default Theme" with yours
  • And now you can customize the appereance of skin - at your own risk :)
